使用本仓库Using This Repository

在 LucheShell 中添加本仓库很简单，只需在命令行执行以下命令：

Adding this repository to LucheShell is simple, just run the following command:

plugins repo lushrepo.lucheshidi.dpdns.org

执行后，LucheShell 会从本仓库中添加插件源。您就可以通过 LucheShell 的插件管理器浏览和安装本仓库中的插件。

After execution, LucheShell will add the plugin source from this repository. You can then browse and install plugins from this repository through LucheShell's plugin manager.

关于本仓库About This Repository

LucheShell Repository 是一个由社区驱动的开源项目，旨在为全球开发者和用户提供便捷的插件分发和管理服务。我们致力于维护高质量的插件库，确保所有文件的安全性和可靠性。

LucheShell Repository is a community-driven open-source project dedicated to providing convenient plugin distribution and management services for developers and users worldwide. We are committed to maintaining a high-quality plugin library with security and reliability.
